# Performance Tests with Locust Tool


https://locust.io/


## Install locust
```
sudo dnf install python3-devel
pip3 install locust
```

## Start locust
```
locust -f <test-file.py>
```

## Run Tests
Launch tests from UI http://localhost:8089/
